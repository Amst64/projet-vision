# projet

Projet du cours Optimisation des performances et architecture des processeurs. 
Le but du projet est d'implémenter un code qui permet de d'obtenir les images avec openCV et les traiter avec 
les filtres median et de Sobel.
On utilise dans un premier temps des fonctions de la bibliothèque openCV, puis coder une version naïve et ensuite une version optimisée

code pour lancer le programme `./projet custom`
